# Allgemeine Abgabeinformationen

Dieses Repo ist als strukturierte M183-Abgabe aufgebaut.

## Struktur

| Pfad | Inhalt |
|---|---|
| `KN-01.md` bis `KN-04.md` | Offizielle Aufgabenstellungen |
| `KN01/` bis `KN04/` | Eigentliche Abgaben mit Nachweisen |
| `ABGABE_STATUS.md` | Aktueller Abgleich: erledigt, teilweise belegt, offen |
| `EC2-Setup.md` | Kurzreferenz fuer die AWS-/Docker-Basis |

## Regeln fuer Nachweise

- Screenshots liegen im jeweiligen KN-Ordner neben der Abgabe-Datei.
- Markdown-Bildlinks bleiben relativ, damit sie lokal und auf GitHub funktionieren.
- Arbeitslogs, lokale Tool-Dateien und Mac-Systemdateien gehoeren nicht ins Repo.
- Offene Punkte werden transparent dokumentiert statt nachtraeglich behauptet.

## Vor der finalen Abgabe

1. `ABGABE_STATUS.md` lesen.
2. Jede KN-Datei im Browser oder in GitHub Preview oeffnen.
3. Fehlende Screenshots gezielt nachreichen.
4. Danach `git status` pruefen und nur relevante Abgabedateien committen.
